# 权威菜谱来源说明

本文档列出菜谱生成时应优先参考的权威来源，按可信度排序。

## 顶级权威来源（优先级A）

### 1. 米其林指南 / Michelin Guide
- **可信度**：⭐⭐⭐⭐⭐
- **特点**：国际顶级美食评鉴标准，菜品精致，技法考究
- **适用**：高端菜品、创意料理、西式烹饪技法
- **搜索关键词**：`米其林 {菜名}` 或 `Michelin {dish name}`

### 2. 舌尖上的中国
- **可信度**：⭐⭐⭐⭐⭐
- **特点**：央视权威纪录片，展现中华传统烹饪精髓
- **适用**：传统中国菜、地方特色菜、非遗烹饪技艺
- **搜索关键词**：`舌尖上的中国 {菜名}`

### 3. 中国烹饪协会
- **可信度**：⭐⭐⭐⭐⭐
- **特点**：国家级行业协会，专业标准制定者
- **适用**：标准化菜品、行业认证配方
- **搜索关键词**：`中国烹饪协会 {菜名} 标准`

### 4. 国宴菜谱
- **可信度**：⭐⭐⭐⭐⭐
- **特点**：钓鱼台国宾馆、人民大会堂等国宴配方
- **适用**：高规格宴请菜品、经典名菜
- **搜索关键词**：`国宴 {菜名}` 或 `钓鱼台 {菜名}`

## 专业来源（优先级B）

### 5. 王刚美食作家
- **可信度**：⭐⭐⭐⭐
- **特点**：专业厨师出身，技法专业，讲解详细
- **适用**：川菜、家常菜、专业技法学习
- **搜索关键词**：`王刚 {菜名}`

### 6. 美食台 / 日食记
- **可信度**：⭐⭐⭐⭐
- **特点**：专业美食视频制作，食材处理精细
- **适用**：视觉呈现要求高的菜品
- **搜索关键词**：`美食台 {菜名}` 或 `日食记 {菜名}`

### 7. 下厨房（评分9.0+）
- **官网**：https://www.xiachufang.com/
- **可信度**：⭐⭐⭐⭐
- **特点**：用户实测验证，大量评论反馈
- **适用**：家庭可操作性强的菜品
- **搜索关键词**：`site:xiachufang.com {菜名}`
- **筛选条件**：仅采用评分9.0以上、做过人数1000+的菜谱

## 来源评估原则

1. **交叉验证**：同一菜品至少参考2个以上来源
2. **实操验证**：优先选择有视频演示的来源
3. **用户反馈**：参考大量用户实际制作的反馈
4. **专业背景**：确认制作者具有专业烹饪背景
5. **时效性**：注意菜谱发布时间，优选近5年内容

## 信息整合策略

当多个来源信息冲突时：
1. 以**更权威来源**为准
2. 如权威性相当，以**更详细的说明**为准
3. 关键步骤采用**多数来源一致**的做法
4. 技巧类信息可**综合多个来源**
